# Level 3 to 4

## Level Goal
> The password for the next level is stored in a hidden file in the 'inhere' directory.

## Comments
This level taught be about reading hidden files in Linux and how to find them using the '-a' parameter on 'ls'

Code Used
------
```shell
ls
cd inhere
ls -a
cat .hidden
```
